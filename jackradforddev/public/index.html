Turn - Reading Tracker

Includes the ability to query the Google Books API to retrieve volume data. Persists user data using SQLite. Also uses local notifications, in-app purchases and Firebase Analytics.

7,000 installs, 320 daily active users




Trait - Habit Tracker

Persists user data using SQLite. Also uses local notifications for habit reminders, in-app purchases and Firebase Analytics.



Both apps, Turn and Trait utilise libraries including: provider, get_it, flutter_local_notifications and sqflite.

Other apps include a water intake tracker, Fluidify, for iOS and Android, and a Wordle clone, Worg, for iOS and Android.
